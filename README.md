# Hackpad

This is a hackpad I built myself. A hackpad is basically a small keypad where every key does whatever you want — shortcuts, macros, or anything else you can think of. I thought it was a really cool idea so I decided to just build one myself.

I started by designing the PCB, which was one of the harder parts since I had to figure out how everything connects. After that I drew the case in CAD so it would actually look like a proper finished product and not just a loose circuit board. The firmware I wrote in C — it runs directly on the hardware and handles all the key inputs.

It took quite a few tries to get everything working together, but in the end I'm really happy with how it turned out. It's one of those projects where you go from just an idea to an actual physical thing you can hold, which is a pretty cool feeling.

## What's in this repo

- `firmware/` — the code that runs on the hackpad
- `pcb/` — the PCB design files
- `cad/` — the 3D files for the case
- `assets/` — images and other files
- `production/` — the files I used to get it manufactured
